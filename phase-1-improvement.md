# Lecsy Phase 1 Improvement Plan

## Essential Improvements

- **1. Add Demo Without Sign-Up** ✔
  - Let users upload a sample file or their own (limit: 2–5 minutes).
  - Place the demo section above the fold on the homepage.

- **2. File Format Support**
  - Clearly indicate support for: MP3, WAV, M4A (audio only).

- **3. Progress Indicator**
  - Show upload progress (e.g., “Uploading... 60%”).
  - Show transcription progress (e.g., “Transcribing... 40%”).
  - Display completion message and transcript.

- **4. Basic Text Editor**
  - Allow users to make minor edits to the transcript before download.

- **5. Download Options**
  - Enable downloads in TXT (plain text), DOCX (Word), and PDF formats.
